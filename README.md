# Playtolia SDK for Unity

Embed Playtolia services to your Unity game with ease.

## Installation

1. Download the latest .unitypackage from the [Releases page](https://github.com/Playtolia/unity-sdk/releases).
2. Import the package into your Unity project by dragging and dropping it into the Unity Editor or using the `Assets > Import Package > Custom Package...` menu option.
3. Select all package contents and click "Import".

## Usage

For detailed usage instructions, refer to the [Playtolia SDK documentation](https://docs.playtolia.com).
